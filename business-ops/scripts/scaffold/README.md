# Scaffold Scripts

Project initialization templates for common patterns.

## Usage

```bash
./scripts/scaffold/saas-api.sh my-project
./scripts/scaffold/consulting-site.sh my-project
./scripts/scaffold/ecommerce.sh my-project
```

## Available Templates

- `saas-api.sh` — REST API + database (Node/Express or Python/FastAPI)
- `saas-frontend.sh` — React/Next.js SaaS frontend
- `consulting-site.sh` — Marketing site for consulting business
- `ecommerce.sh` — Shopify/WooCommerce setup scripts
- `data-pipeline.sh` — ETL/data processing pipeline
- `chrome-extension.sh` — Browser extension starter
